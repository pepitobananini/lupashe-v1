# Public Assets

Coloca aquí el logo de lupaSHE.

## Logo

1. Coloca tu imagen del logo como: `logo.png`
2. El logo debe estar en formato PNG con fondo transparente (recomendado)
3. Tamaño recomendado: mínimo 400x200px para buena calidad

El componente Logo buscará automáticamente `/logo.png` y si no existe, mostrará un logo SVG basado en la descripción del logo.
